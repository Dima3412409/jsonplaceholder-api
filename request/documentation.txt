Документация к тестам

1. Get_posts.py
Цель: Проверка GET-запросов для получения всех постов.
•	Функция test_get_posts: Проверяет корректность кода ответа (200 OK), заголовка Content-Type, структуру данных (должен быть список словарей с полями userId, id, title, body), а также проверяет первые несколько постов на соответствие ожидаемым значениям
•	Функция test_get_invalid_post: Параметризованный тест, проверяющий некорректные URL и их статус-коды

2. Get_posts_id.py
Цель: Проверка GET-запросов для получения конкретного поста по ID.
•	Функция test_get_posts: Проверяет корректность ответа для существующего поста (ID = 1), проверяя код ответа, структуру данных и содержание поста
•	Функция test_get_post_parametrized: Параметризованный тест, проверяющий запросы к различным постам по ID и их статус-коды
•	Функция test_get_post_content: Параметризованный тест, проверяющий корректность заголовка (title) для конкретных постов

3. Post_posts.py
Цель: Проверка POST-запросов для создания нового поста.
•	Функция test_post_posts: Проверяет корректность ответа при создании нового поста, его структуру данных и обработку пустого запроса
•	Функция test_post_parametrized: Параметризованный тест, проверяющий различные типы запросов и соответствующие статус-коды. Проверяются корректные запросы и некорректные форматы данных

4. PUT_posts_id.py
Цель: Проверка PUT-запросов для обновления существующих постов.
•	Функция test_put_posts: Проверяет корректность ответа при обновлении поста (ID = 1), и проверку структуры и содержания ответа
•	Функция test_put_post_parametrized: Параметризованный тест, проверяющий запросы на обновление постов с различными ID и payload, а также обработку некорректных форматов данных

5. DELETE_posts_id.py
Цель: Проверка DELETE-запросов для удаления постов.
•	Функция test_delete_post: Проверяет корректность ответа при удалении поста (ID = 1), включая проверку структуры ответа
•	Функция test_delete_post_parametrized: Параметризованный тест, проверяющий запросы на удаление постов с различными ID и их статус-коды. Также проверяет, что после удаления поста его нельзя получить
